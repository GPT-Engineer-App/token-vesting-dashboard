# token-vesting-dashboard

A dashboard of how many tokens have vested.

With inputs for each individual:
 - Allocation
 - Schedule (start and duration)
 - Cliff

Visualized with a stacked barchart

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/token-vesting-dashboard.git
cd token-vesting-dashboard
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
